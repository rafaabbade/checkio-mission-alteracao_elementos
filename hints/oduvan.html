Como atualizar um valor em uma lista?

Para atualizar um valor específico em uma lista, você pode acessar o elemento pelo índice e atribuir um novo valor a ele. 

precos = [100.0, 200.0, 300.0]
indice = 1
precos[indice] = novo_preco

Como arredondar um valor para uma casa decimal?

Para garantir que o valor atualizado seja arredondado para uma casa decimal, você pode usar a função round(). Por exemplo: 

valor = 220.456
valor_arredondado = round(valor, 1)
print(valor_arredondado)  # Saída: 220.5
Isso é útil para manter a precisão no resultado final.
